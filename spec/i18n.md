# Internationalization (i18n)

## Supported Languages

| Code | Language |
|------|----------|
| `en` | English (source language — no `.qm` file needed) |
| `fi` | Suomi |
| `sv` | Svenska |
| `da` | Dansk |
| `nb` | Norsk |
| `es` | Español |
| `de` | Deutsch |
| `pt` | Português |

---

## File Layout

| Path | Purpose |
|------|---------|
| `translations/libreai_XX.ts` | Qt XML source — edit these to update translations |
| `translations/libreai_XX.qm` | Compiled binary — committed, regenerated by `lrelease` during build |
| `translations/libreai_translations.qrc` | Embeds all `.qm` files at `:/i18n/` |

---

## Qt Conventions Used

| Location | Pattern | Reason |
|----------|---------|--------|
| `ChatWindow`, `ConfigDialog` | `tr("...")` | Both are `Q_OBJECT` classes |
| `CMInterceptor` | `QCoreApplication::translate("CMInterceptor", "...")` | No `Q_OBJECT` macro |

Both window classes implement `retranslateUi()` called from `changeEvent(QEvent::LanguageChange)` so labels update immediately when `Config::applyLanguage()` swaps the active `QTranslator` — no window restart required.

---

## How applyLanguage() Works

```cpp
// Config::applyLanguage()
// 1. Remove and delete the previous QTranslator (if any)
// 2. Return early for "en" — source strings are English
// 3. Load ":/i18n/libreai_<lang>.qm" into a new QTranslator
// 4. QApplication::installTranslator(translator)
//    → Qt broadcasts QEvent::LanguageChange to all top-level widgets
//    → ChatWindow and ConfigDialog catch it and call retranslateUi()
```

---

## LibreOffice Menu Items

`Addons.xcu` menu labels use `xml:lang` attributes and follow **LibreOffice's own UI language**, not LibreAI's language setting. They are not affected by `applyLanguage()`.

---

## Adding a New Language

1. Copy an existing source file:
   ```bash
   cp translations/libreai_fi.ts translations/libreai_XX.ts
   ```

2. Translate every `<translation>` element in `translations/libreai_XX.ts`.

3. Add the language to `kLanguages[]` in `src/ConfigDialog.cpp`:
   ```cpp
   {"XX", "NativeName"},
   ```

4. Add the compiled file to `translations/libreai_translations.qrc`:
   ```xml
   <file>libreai_XX.qm</file>
   ```

5. Add `xml:lang="XX"` entries for menu item labels in `Addons.xcu`.

6. Compile and test:
   ```bash
   lrelease translations/libreai_XX.ts
   bash build.sh --install
   ```
