LibreAI — AI writing assistant for LibreOffice

A native floating chat window with a dark VS Code-inspired theme that brings
AI models directly into your document workflow. Works in Writer, Impress,
and Calc. No Java required.

Features:
• 5 AI providers: Ollama (local), OpenAI, Anthropic Claude, xAI Grok, Google Gemini
• Markdown rendering — responses display with headers, bold, code blocks, and lists
• Rich text apply — transfers headings, bold, italic, and lists as native Writer styles
• Right-click "Ask from AI" to open with selected text pre-loaded
• Multi-turn conversation history
• Rewrite selected text with a single click
• Secure credential storage (OS keychain — never stored as plaintext)
• Ollama remote auth: None, Basic Auth, or API Key header
• Comprehensive logging with configurable level and size limit
• UI available in 8 languages

Requirements:
• LibreOffice 7.0 or newer
• For Ollama: Ollama running at http://localhost:11434
• For cloud providers: valid API key (OpenAI, Anthropic, xAI, or Google AI Studio)
